# 42-Fractol
Ce projet consiste à créer un petit logiciel d’exploration fractale.

    125/100 :)

## compile

    make
  
## run

    ./fractol <mandelbrot/julia/burningship>

## keys

    MOVE : up down left right
    ZOOM : + - (numeric)
    DEEP : pageup pagedown
    RESET : r
    PSYCHADELIC MODE : 2
    AUTO ZOOM : 1
